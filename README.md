# 🚀 React Projects - Chai aur Code Collection

A comprehensive collection of **10 real-world React projects** from the "React JS by Chai aur Code" course. All projects are built with React 18, Vite, and modern JavaScript, organized in a monorepo structure.

## 📋 Projects Included

### 1. **Counter App** (`01-counter-app`)
A simple counter application with increment, decrement, and reset functionality.
- ✨ Features: State management with hooks
- 🎨 Beautiful gradient UI with animations
- **Port:** 3000

### 2. **Password Generator** (`02-password-generator`)
Generate secure passwords with customizable options.
- 🔐 Features: Length control, special characters, numbers
- 📋 Copy to clipboard functionality
- **Port:** 3001

### 3. **Currency Converter** (`03-currency-converter`)
Convert between different currencies in real-time.
- 💱 Features: Live exchange rates API
- 🔄 Swap currencies functionality
- 📊 Support for multiple currencies
- **Port:** 3002

### 4. **Todo App** (`04-todo-app`)
Complete todo list with local storage persistence.
- ✅ Features: Add, delete, mark complete
- 💾 Local storage support
- 📈 Progress tracking
- **Port:** 3003

### 5. **Weather App** (`05-weather-app`)
Get weather information for any city.
- 🌍 Features: Search by city
- 🌤️ Detailed weather metrics
- 📊 Display humidity, wind speed, UV index
- **Port:** 3008

### 6. **Color Picker** (`06-color-picker`)
Advanced color picker with multiple format support.
- 🎨 Features: HEX, RGB, HSL formats
- 🌓 Dark/Light theme toggle
- 🎯 Color conversion
- **Port:** 3004

### 7. **Calculator** (`07-calculator`)
Full-featured calculator with operation history.
- 🧮 Features: Basic arithmetic operations
- 📜 Calculation history
- 🎯 Percentage and decimal support
- **Port:** 3005

### 8. **BMI Calculator** (`08-bmi-calculator`)
Health-focused BMI calculation tool.
- ⚖️ Features: Height/Weight input
- 📊 Category classification
- 📈 Visual progress indicators
- **Port:** 3006

### 9. **Background Changer** (`09-background-changer`)
Dynamic background customization with multiple options.
- 🎨 Features: Color picker, presets, gradients
- 🖌️ Border styling options
- 🌓 Theme toggle
- **Port:** 3007

### 10. **Tic Tac Toe** (`10-tic-tac-toe`)
Classic game with move history.
- 🎮 Features: Two-player gameplay
- 🏆 Winner detection
- 📜 Move history tracking
- **Port:** 3009

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/himanshi01-stack/react-counter-app.git
cd react-counter-app
```

2. Install dependencies for a specific project:
```bash
cd 01-counter-app
npm install
```

## 🚀 Running Projects

Each project runs on a different port for easy development:

```bash
# Counter App
cd 01-counter-app && npm run dev

# Password Generator
cd 02-password-generator && npm run dev

# Currency Converter
cd 03-currency-converter && npm run dev

# Todo App
cd 04-todo-app && npm run dev

# Weather App
cd 05-weather-app && npm run dev

# Color Picker
cd 06-color-picker && npm run dev

# Calculator
cd 07-calculator && npm run dev

# BMI Calculator

# Background Changer
```

## 🏗️ Building for Production

```bash
cd [project-folder]
npm run build
npm run preview
```

## 💡 Learning Concepts

These projects cover essential React concepts:

- **Hooks:** useState, useEffect, useCallback
- **State Management:** Component state, local storage
- **API Integration:** Fetching data from APIs
- **Event Handling:** Click events, keyboard events, form inputs
- **Conditional Rendering:** Show/hide elements based on state
- **List Rendering:** Mapping over arrays
- **Styling:** CSS, gradients, animations
- **Component Composition:** Reusable components
- **Props Drilling:** Passing data through components
