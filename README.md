# React Counter App

A simple counter application built with React and Vite.

## Features

- Increment counter
- Decrement counter
- Reset counter
- Beautiful gradient UI with smooth animations
- Responsive design

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **CSS3** - Styling with gradients and animations
- **JavaScript (ES6+)** - Modern JavaScript

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

The app will open at `http://localhost:3000`

### Building for Production

```bash
# Build the app
npm run build

# Preview production build
npm run preview
```

### Linting

```bash
# Run ESLint
npm run lint
```

### Testing

```bash
# Run tests with Vitest
npm test
```

## Usage

The counter app is simple to use:

1. Click **Increment** to increase the counter
2. Click **Decrement** to decrease the counter
3. Click **Reset** to return the counter to 0
